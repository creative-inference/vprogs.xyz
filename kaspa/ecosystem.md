# The Kaspa Ecosystem

Kaspa's lightning-fast BlockDAG architecture is the foundation for a rapidly expanding ecosystem of applications, Layer 2 networks, and secure infrastructure.

## Programmability and dApps

Kaspa's approach to programmability is unique. Rather than relying on a single monolithic model like the EVM, Kaspa offers a set of powerful building blocks tailored to different developer needs, particularly concerning concurrent execution:

- **L1 Covenants:** Enforce powerful asset-native rules and state machines directly on the Kaspa base layer. Perfect for custom transfer policies, issuance schedules, and custody rules.
- **Inline ZK Proofs:** A dedicated path for privacy and custom validation, allowing individual user actions to privately prove their own validity. This enables private apps and custom account models.
- **Based Apps:** The preferred solution for dApps needing shared state and concurrency. Developers can build in Rust within a managed environment that seamlessly settles back to the Kaspa Layer 1.
- **Full vProgs:** Kaspa's future vision for a completely interconnected and composable app ecosystem, where independent applications can seamlessly interact and compose with each other.

## Kasplex EVM Layer 2

The Kasplex network brings EVM (Ethereum Virtual Machine) compatibility to Kaspa, unlocking smart contracts and decentralized applications. Kasplex is experiencing a massive surge in activity, processing a growing volume of daily transactions and expanding the possibilities of what can be built on the Kaspa network.
