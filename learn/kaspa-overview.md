# What is Kaspa?

Kaspa is a revolutionary proof-of-work cryptocurrency built on a BlockDAG (Block Directed Acyclic Graph) architecture. Kaspa originator Yonatan Sompolinsky describes it as **"real-time bitcoin"**—it delivers the robust security guarantees of Bitcoin but in seconds instead of an hour, by decentralizing each consensus round.

## Unmatched Scalability

Kaspa solves the traditional blockchain trilemma by incorporating all valid blocks simultaneously.
- **10 Blocks Per Second:** Thanks to the Crescendo hardfork, the network processes 10 blocks every single second.
- **No Wasted Work:** Unlike traditional blockchains, the DAG includes all honest blocks and wastes no proof-of-work, translating the full mining power of the network into usable throughput.

## A Fair Foundation

True to the cypherpunk ethos, Kaspa launched fairly in November 2021:
- Zero pre-mine
- Zero pre-sale
- No venture capital allocations
- No foundation treasury

It remains a pure, decentralized, open-source project driven by its community.
